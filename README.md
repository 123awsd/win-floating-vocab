# 桌面背单词 (W2R Cattoon)

中文说明 | [English](README_EN.md)

一个 Windows 桌面悬浮背词工具，支持自定义词表、自动/手动切换、英日朗读、主题与字体调整。

## 功能亮点
- 悬浮窗背词：不打断当前工作流
- 词库切换：自动识别英/日词表并切换朗读语言
- 多种交互：双击、空格、方向键、右键菜单
- 外观可调：主题色、单词颜色、数字颜色、字体、透明度滑条
- 学习进度：记录今日计数，支持收藏词条

## 环境要求
- Windows 10 / 11
- Python 3.10+

## 快速开始
```powershell
cd catword
python -m pip install -r requirements.txt
python W2R.py
```

## 使用教程
### 基础操作
- 拖动窗口：鼠标左键按住悬浮窗
- 下一个词：双击窗口 / `Space` / `Right`
- 上一个词：`Left`
- 再次朗读当前词：`R`
- 缩放窗口：鼠标滚轮
- 打开设置：窗口内右键

### 右键菜单（常用）
- `播放模式`：手动 / 自动
- `播放顺序`：顺序 / 随机
- `播放速度`
- `朗读当前词`、`自动朗读`
- `清零今日计数`
- `复制该词`、`收藏该词`
- `单词颜色`、`数字颜色`、`主题色`、`字体`、`透明度滑条`
- `切换词库`
- `保存配置`、`恢复配置`

## 自定义词表
将 `.txt` 词表放到 `catword/` 目录，然后在右键菜单 `切换词库` 选择。

推荐格式（每行一个词条，Tab 分隔）：
```text
reinforcement learning	强化学习（RL）
policy optimization	策略优化
```

## 日语朗读与语音包安装
程序会在切换词库时自动识别语言；若词表为日语但系统无日语语音包，将回退到英语语音。

### 安装日语语音包（Windows 11）
1. 打开 `设置 -> 时间和语言 -> 语言和区域`
2. 添加语言 `日语`
3. 进入 `日语 -> 语言选项`
4. 安装 `语音 (Speech)` 组件
5. 重启系统后重新打开程序

### 安装日语语音包（Windows 10）
1. 打开 `设置 -> 时间和语言 -> 语言`
2. 添加 `日本語`
3. 进入语言选项安装 `语音`
4. 重启系统后重新打开程序

说明：日语语音包是 Windows 系统组件，不能可靠地随本项目一起打包分发到项目目录中。
