# 桌面背单词（W2R Cattoon）使用教程

这是一款 Windows 桌面悬浮背词工具。  
本 README 以“直接用起来”为主。

## 1. 先准备环境（只做一次）

```powershell
cd catword
python -m pip install -r requirements.txt
```

## 2. 启动程序

```powershell
python W2R.py
```

启动后会出现悬浮窗，默认显示英文词和中文释义。

## 3. 第一次使用怎么操作

- 拖动位置：鼠标左键按住窗口拖动
- 上一个词：按键盘 `←`
- 下一个词：双击窗口，或按键盘 `空格` / `→`
- 重新朗读当前词：按键盘 `R`
- 缩放窗口：鼠标滚轮向上/向下
- 右键菜单：在窗口上右键，打开全部设置

补充说明：

- `←` / `→` 支持在已浏览的单词之间前后切换
- 即使在乱序模式下，按 `←` 也会回到刚刚看过的词

## 4. 右键菜单怎么用（最常用）

右键后你会看到这些功能：

- `切换模式`：手动切换 / 自动切换
- `切换顺序`：顺序播放 / 乱序播放
- `播放速度`：调自动切词速度
- `朗读当前词`、`自动朗读`
- `清零今日计数`
- `复制该词`、`收藏该词`
- `单词颜色`、`数字颜色`、`主题色`、`字体`、`透明度`
- `切换词库`
- `保存配置`、`恢复配置`
- `退出`

## 5. 如何使用你自己的词表（重点）

### 推荐格式：TAB 分隔

每行一个词条：

```text
reinforcement learning	强化学习（RL）
policy optimization	策略优化
```

要求：

- 左侧是英文词/短语，右侧是中文释义
- 文件编码建议 `UTF-8`
- 一行一个词条，不要空太多无效行

### 放到哪里

把你的词表 `.txt` 放到 `catword/` 目录里。  
然后在程序里右键 `切换词库` 选择对应文件。

## 6. 自带示例词表

位置：

- `catword/高考3500词汇表_乱序.txt`
- `catword/文献术语精选_280_乱序.txt`

你可以直接切换到这些词表体验。

## 7. 常见问题

### Q1：点击朗读没声音

- 先检查系统语音是否可用
- 不同电脑可用音色会不一样
- 也可以直接按键盘 `R` 重新朗读当前单词

### Q2：词表切换后没变化

- 检查 txt 是否是 UTF-8
- 检查是否为 `term<TAB>explanation` 格式
- 重启程序后再试一次

### Q3：窗口太大/太小

- 用鼠标滚轮缩放
- 或在右键菜单调字体、透明度

## 8. 打包（可选）

如果你要自己打包 exe：

```powershell
pyinstaller --noconfirm --clean W2R_Cattoon_PySide6.spec
```
